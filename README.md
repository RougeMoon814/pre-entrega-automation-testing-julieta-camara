
# Proyecto de Automatización de Saucedemo

## Instalación

- pip install pytest
- python -m pip install pytest-html

## Tecnologias Requeridas
- Python3 com lenguaje principal
- Pytest para estructura de testing
- Selenium WebDriver para automatización

## Reporte
- pytest test/test_saucedemo.py -v --html=reporte.html

### Reporte Alternativo
- pytest test/test_login.py -v --html=reporte1.html
- pytest test/test_catalogo.py -v --html=reporte2.html
- pytest test/test_carrito.py -v --html=reporte3.html
